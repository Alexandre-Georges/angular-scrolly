$draggerProvider (service in module scrolly )

Description

Used for configuring drag options.

Methods

  • maxTimeMotionless(newTime)

    Sets/gets the maximum time a user can be motionless, in milliseconds, before the user is counted as 'not actively dragging' anymore.

    Parameters

    • newTime(optional) – {number=} –

      Sets the maximum time, milliseconds.

    Returns

    {number} –

    maxTimeMotionless The current maximum time motionless value.

  • minDistanceForDrag(newDistance)

    Sets/gets the minimum distance the user needs to move his finger before it is counted as a drag.

    Parameters

    • newDistance(optional) – {number=} –

      Sets the minimum distance value.

    Returns

    {number} –

    minDistanceForDrag Current minimum distance value.
