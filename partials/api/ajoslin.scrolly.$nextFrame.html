$nextFrame (service in module ajoslin.scrolly )

Description

A service to wrap window.requestAnimationFrame, or a fallback if it is not available.

The main reason this is in a service is for ease of mocking it during tests.

Usage

$nextFrame(callback);

Parameters

  • callback – {function} –

    Callback to call when the DOM has redrawn - when the next frame is ready.

Returns

{number} –

requestId Unique id identifying this request, to be passed to window.cancelAnimationFrame.
